About Me

This is a personal app designed for remote access to a greenhouse from anywhere.

This hobby project is an experiment in IoT. I wanted to challenge myself by developing, deploying, and releasing a web page on my own. The app features a simple NodeMCU project that collects and displays sensor data from my greenhouse. It also includes control buttons and configurable settings. My goal is to use the greenhouse to grow mushrooms. As a newcomer to both IoT and mushroom cultivation, I'm open to any suggestions for improvement.

Project Hardware:
  • NodeMCU v3 Lua
  • 2 DHT sensors
  • 2 relays
  • 12V pump
  • 12V fan
Project Software:
  • NodeMCU firmware with Lua: https://nodemcu-build.com/
  • Angular for frontend
  • Spring Boot for backend
Project Capabilities:
  • humiThreshold: Humidity threshold to enable the pump (default: 75%)
  • pumpWorkInterval: Interval for pump activation (default: 5 seconds)
  • fanWorkInterval: Interval for fan operation
  • fanWorkPeriodInterval: Fan operation period. Example: 10 minutes every 20 minutes.
  • sensorDataInterval: Interval for sending sensor data to backend
  • sensorReadInterval: Interval for reading data from sensors
Project Plans:
  • Add CO2 sensor
  • Create a PCB board
  • Experiment with growing different types of mushrooms
